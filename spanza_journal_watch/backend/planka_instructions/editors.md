## Inviting reviewers

The Journal Watch editorial backend can be used to invite reviewers to Planka. An invited reviewer:

- Receives an email with a login/sign up link
- Is added to the Issue's Planka project
- Is assigned edit permissions to the Reviews board

It is encouraged that reviewers use Planka to edit and submit their articles so that:

1. Formatting remains consistent when transferred to the website
2. The Journal Watch backup system can track/restore changes if necessary
3. To assist regional coordinators to keep an overview of the readiness of their issue for publication
4. To promote collaboration and interaction between reviewers (should they so wish)

## Choosing articles

Articles can be pulled from watched journals using PubMed and other sources. This process is done automatically by the Journal Watch backend.

You may filter these articles to the months you expect to cover. Journal by journal, you can 'stage' (select) articles that might be of interest to reviewers. These can then be pushed to Planka, where they will appear in the 'Candidates' category.

Reviewers may also add their own custom cards for reviews not listed. These reviews can be consolidated with the Journal Watch database at import time.

## Editorial pass

Most editing will be done in the Journal Watch backend. However, Planka provides a good opportunity to interact with the reviewer if there are suggested changes.

The main purpose of the editorial process is:

- To pull external articles into Journal Watch where necessary
- To add categorisation tags/verify suggested tags as necessary
- Look over reviews for structure/spelling/grammar

Edits to content should be judicious and preferably minimal where possible

## Escalation protocol

Journal Watch applies the principle that all reviews submitted are published, where possible.

If there are concerns about content that cannot be resolved between regional coordinator and reviewer, please escalate to the Journal Watch chair: [journalwatch@spanza.org.au](mailto:journalwatch@spanza.org.au)
